프론트엔드 전향 이유

기존에는 백엔드 개발자로서 데이터 처리와 비즈니스 로직을 주로 담당했습니다. 보여지는 화면을 직접 만들고 눈에 보이는 결과에 더 큰 성취감을 느껴 프론트에 흥미를 가지게 되었습니다.

내가 생각하는 프론트엔드

프론트엔드는 UI를 그리는 작업이 아니라, 서비스의 구조와 데이터 흐름을 고려해 사용자가 불편함 없이 이용할 수 있도록 조율하는 역할이라고 생각합니다.

보유 스킬

HTML, CSS, JavaScript, React, Bootstrap, TailwindCSS
(Java, Oracle, PL/SQL, MVC구조 이해 가능)

“Design and code evolve together.”

백엔드 개발 경험을 바탕으로 API와 데이터 흐름을 이해하고, 사용성과 유지보수성을 고려한 프론트엔드 구현을 지향합니다.

인터랙션 하나부터 일관된 UI 규칙, 접근성까지 고려한 개발을 중요시 하며 AI를 적극 활용해 빠르고 완성도 높은 개발을 목표로 합니다.
