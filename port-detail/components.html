SJ's UI Components

제가 주로 사용하는 UI 컴포넌트 스타일을
재사용 가능하게끔 기준을 정리한 페이지입니다.

Buttons

주요 액션을 위한 버튼 컴포넌트입니다.
Primary 버튼은 한 화면에 1개 사용을 권장합니다.
Ghost 버튼은 보조 액션이나 텍스트 링크 대체용으로 사용합니다.

Badges

상태나 속성을 간단히 표시하기 위한 컴포넌트입니다.
정보 전달용이며, 클릭 액션에는 잘 사용하지 않습니다.
